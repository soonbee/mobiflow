# spec-backlog (active)

refine phase의 사전·승급 게이트(또는 별도 직접 추가 스킬)에서 등록된 항목 중 `pending` 상태로 다음 spec 라운드 검토를 기다리는 것들이 모이는 파일.

## 동작 (요약)

- **신규 등록** — `refine` 스킬이 `pending` 상태로 본 파일에 항목 추가
- **terminal 전이** — `accepted-v<버전>` / `rejected` / `superseded`로 결정되면 본 파일에서 제거하고 `spec-backlog-archive.md`로 이동
- **③번 우회는 본 파일을 거치지 않음** — refine 게이트의 ③번 인라인 진행은 archive에 직행

각 항목의 스키마와 lifecycle은 `refine` 스킬 문서를 참조한다. 본 파일은 `refine` 스킬이 자동 관리하므로 수동 편집은 권장하지 않는다.
